===========================
How to Use 5-text_indentation.py
===========================

This module defines the function ``def text_indentation(text):``.

USAGE
-----

	>>> text_indentation = __import__('5-text_indentation').text_indentation

	<BLANKLINE>
	How are you?
	<BLANKLINE>
	Currently, we are watching you.
	<BLANKLINE>
